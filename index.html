theme-color demo Seção 01 — Azul Noite

Role a página
e veja a magia.

Cada seção muda a cor da barra do navegador no mobile. Abra no Safari (iOS) ou Chrome (Android) para ver o efeito.

<meta name="theme-color" content="#0f1b2d">

Seção 02 — Verde Esmeralda

A barra agora
é verde.

O JavaScript detecta qual seção está visível via IntersectionObserver e atualiza a meta tag dinamicamente.

<meta name="theme-color" content="#052e16">
Seção 03 — Creme

Funciona em
tons claros.

A cor pode ser qualquer valor CSS válido — hex, rgb, hsl. Aqui usamos um off-white quente para mostrar que também funciona em temas claros.

<meta name="theme-color" content="#faf8f3">
Seção 04 — Roxo Profundo

Só uma linha
de HTML.

A meta tag base fica no <head>. O JS apenas troca o atributo content conforme o scroll — nenhuma biblioteca necessária.

<meta name="theme-color" content="#1e0a3c">
Seção 05 — Vermelho

Simples assim.

Funciona no Safari iOS 15+ e Chrome Android. Navegadores que não suportam simplesmente ignoram a tag — zero risco.

<meta name="theme-color" content="#1c0a0a">
